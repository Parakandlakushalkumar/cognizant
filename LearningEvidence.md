# DN5 Upskilling Learning Evidence

This file summarizes the practical files added inside the DN5 Upskilling folder. Each folder contains small examples that prove fundamental and intermediate understanding of the topic.

## CSharp

- `Assignments/CSharpFundamentals.cs` shows variables, data types, conditions, switch expressions, collections, LINQ basics, and exception handling.
- `HandsOn/StudentGradeCalculator.cs` shows records, lists, methods, average calculation, and simple reporting logic.

## HTML5

- `Assignments/semantic-portfolio.html` shows semantic page structure, navigation, sections, articles, tables, and accessibility basics.
- `HandsOn/form-validation-demo.html` shows form controls, labels, required fields, email validation, number limits, select, and textarea.
- `MiniProject/personal-learning-page.html` shows a small complete HTML5 learning journal page.

## CSS3

- `Assignments/responsive-profile-card.html` and `responsive-profile-card.css` show selectors, box model, card styling, flexbox, hover-ready structure, and responsive design.
- `HandsOn/layout-practice.html` and `layout-practice.css` show page layout using Grid, repeated items using Flexbox, hover states, and mobile adjustment.

## JavaScript

- `Assignments/array-string-practice.js` shows variables, arrays, template strings, map, filter, functions, objects, and destructuring.
- `HandsOn/todo-app.html` shows DOM selection, event handling, application state with arrays, rendering, input validation, and removing tasks.

## SQL

- `Assignments/library-schema.sql` shows database creation, tables, primary keys, foreign keys, constraints, inserts, and joins.
- `HandsOn/query-practice.sql` shows select, where, order by, group by, join, update, and transaction basics.
